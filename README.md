## Rainy Days

Rainy Days is an interactive online store for outdoor clothing built with HTML, CSS and JavaScript.  
The project displays products from the Rainy Days API and allows users to browse products, view details, add items to a cart, and complete a checkout flow.

The goal of the project was to practice working with APIs, dynamic JavaScript functionality, and building a small e-commerce experience.

## Features

- View a list of products fetched from the Rainy Days API
- View detailed information about each product
- Add and remove items from the shopping cart
- Cart badge that shows the number of items in the cart
- Checkout page with order summary
- Order confirmation page
